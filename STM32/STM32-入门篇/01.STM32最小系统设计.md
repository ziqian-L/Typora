### 一、什么是最小系统

MCU最小系统：一个MCU在能正常工作，正常下载程序的前提下的最简电路

 ### 二、STM32最小系统：

（1）组成部分

- 供电
- 复位
- 时钟：外部晶振（两个）
- Boot启动模式选择
- 下载电路（串口/JTAG/SWD）
- 后备电池

（2）供电电路

**VDD和VDDA之间一般连接一个简单的低通滤波器即可（RC、兀型均可）**

| 名称         | 典型值 |
| :----------- | ------ |
| VDD数字电源  | +3.3V  |
| VSS数字地    | 0      |
| VDDA模拟电源 | +3.3V  |
| VSSA模拟地   | 0      |
